# 764 Group Online Threat Briefing (Public Awareness Tool)

This repository contains a standalone, browser-based briefing tool summarizing publicly available information about online grooming, extortion, and coordinated abuse patterns associated with networks commonly referred to as the *764 group*.

The purpose of this project is to support:

- Parents and caregivers  
- Educators and school administrators  
- Community leaders and youth workers  
- Local law enforcement and first responders  

by providing a clear, structured overview of the tactics, platforms, and warning signs associated with online predatory behavior.

---

## Features

- Comprehensive overview of publicly discussed 764 group activity  
- Recruitment, grooming, extortion, and escalation phases  
- Documented public cases (summarized for safety and privacy)  
- High-risk platform intelligence  
- Prevention & response strategies for parents, educators, and law enforcement  
- Built-in print mode for generating PDF handouts  
- Zero installation: open `index.html` in any modern browser  

---

## Sources

This briefing tool organizes and summarizes information drawn from:

- Interviews and public discussions, including **The Shawn Ryan Show**  
- U.S. Department of Justice press releases  
- FBI child protection resources  
- IC3 Public Service Announcements (PSAs)  

All referenced information is publicly available.

---

## How to Use

1. Download `index.html`  
2. Open it in any modern web browser  
3. Navigate through sections using the sidebar  
4. Use the built-in **Print/PDF** button to generate physical copies or distribute as needed  

No external dependencies or server components are required.
